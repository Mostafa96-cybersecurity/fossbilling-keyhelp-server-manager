# FOSSBilling KeyHelp Server Manager

KeyHelp provisioning module for FOSSBilling.

## Features

- Automatic account creation
- Suspension / Unsuspension
- Package change
- Domain synchronization
- API caching
- Retry logic

## Installation

Copy the file:

KeyHelp.php

to:

library/Server/Manager/

inside your FOSSBilling installation.

## Version

1.0.0
